Tvarit - AWS DevOps automation

Tvarit is used by DevOps teams using familiar tool chains such as Maven to automate deployments to AWS.

Scale & Speed

Tvarit's primary goal is to reduce or eliminate all manual steps between committing code and serving it to customers. It is built on top of infinite scale of Amazon Web Services and wide feature range of Wildfly AS.

Read More

Fork it on GitHub

Tvarit is open source and will be made available on Maven Central in Feb 2016. You can fork it and improve it - or just ask us to do it for you. Getting started is easy and support is available through GitHub issues.
